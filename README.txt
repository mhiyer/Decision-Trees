Project Description:
Build an ID3 decision tree from scratch using Python3- the libraries math, numpy, pandas are used.
The output is binary - with values either 0 or 1. The attributes may take on any number of values , e.g. 'Sky' may be 'Sunny', 'Rainy', 'Overcast' etc.

Codes:
1) get_root_node.py: Get the root node given a pandas data structure. The aim is to build a decision tree using ID3.
